# VAPEコイル抵抗値計算機

VAPEのコイル作成時に必要な抵抗値を簡単に計算できるWebアプリケーションです。

## 機能

- ワイヤー素材の選択（カンタル、ニクロム、ステンレス）
- ワイヤーゲージ（AWG）の選択
- コイル内径の設定
- 巻き数の設定
- シングル/デュアルコイルの選択
- リアルタイム計算
- 必要なワイヤー長の表示

## 使用方法

1. ワイヤー素材を選択
2. ワイヤーゲージ（太さ）を選択
3. コイルの内径を入力
4. 巻き数を入力
5. コイル数を選択
6. 自動的に抵抗値が計算されます

## 技術仕様

- HTML5
- CSS3 (レスポンシブデザイン)
- Vanilla JavaScript
- 静的サイト（サーバー不要）

## ローカル実行

1. リポジトリをクローン
2. `index.html`をブラウザで開く
